# 避坑指南与 Debug 日志 (Troubleshooting)

## 1. 渲染：贪婪网格化 (Greedy Meshing) 几何爆炸
*   **症状**：屏幕出现跨越空间的黑色拉伸三角形或闪烁碎片。
*   **原因**：属性对齐错误、坐标变量污染以及绕序错位。
*   **修复**：当前版本已回滚至 **Naive Surface Culling** 稳定版。后续尝试需引入严格的长度对齐校验。

## 2. 物理：出生点判定歧义
*   **症状**：在未加载区域出生时，玩家偶尔被传送到 Y=0。
*   **修复**：将 `getHighestBlock` “未找到”状态显式修改为返回 **`null`**，并在 `findSafeSpawn` 中增加判空逻辑。

## 3. 交互：背包开关回路冲突
*   **症状**：按 E 键打开背包时，UI 一闪而过自动关闭。
*   **修复**：引入 `isOpeningInventory` 标记位。确保 `unlock` 监听器能清晰区分“按 ESC 退出”和“按 E 打开背包”两种状态。

## 4. 交互：背包满载丢物品
*   **症状**：背包满了，挖掉的方块或关闭 UI 时的手持物品直接消失。
*   **修复**：
    1. 挖掘前校验 `addItem`。若失败则拦截破坏流程。
    2. 关闭背包时校验 `addItem`。若失败则通过 `controls.unlock()` 强制拦截 UI 关闭。

## 5. 存储：未加载区块修改丢失
*   **症状**：修改了视距外的方块（例如通过指令或爆炸），重新进入后发现没有保存。
*   **原因**：`setBlock` 仅在 `chunk` 存在于内存时执行。
*   **修复**：重构 `WorldManager.setBlock`，将数据库持久化逻辑 (`saveChunkDelta`) 移出 `if (chunk)` 判断，实现全域持久化。

## 6. 世界生成：地形非确定性漂移
*   **原因**：`simplex-noise` 未固定种子。
*   **修复**：引入 `alea` 确定性随机生成器并同步给主线程与 Worker。

## 7. 交互：无限手长
*   **修复**：在射线检测中增加 `if (intersect.distance > MAX_REACH) return;` 限制（默认 5 格）。
