# 避坑指南与 Debug 日志 (Troubleshooting)

## 1. 渲染：深度排序错乱 (深度测试)
* **症状**：远处的方块渲染在近处的方块之上。
* **原因与修复**：确认 `material.depthWrite` 和 `depthTest` 设置正确。区分透明（水、玻璃）与非透明材质的 `transparent` 属性。

## 2. 物理：玩家落地后剧烈抖动
* **症状**：触地瞬间玩家视角狂跳。
* **原因与修复**：碰撞修复（Position Correction）后未清理 Y 轴速度。修复方法：`if (hitGround) velocity.y = 0;`。

## 3. 物理：出生卡墙（上帝视角正常，生存无法移动）
* **症状**：刚进游戏玩家无法走动。
* **原因与修复**：
    1. **高度重叠**：将玩家置于最高方块上时，脚部高度计算未考虑玩家碰撞箱。修复：`camera.position.y = y + 1 + eyeHeight;`。
    2. **高度加载顺序**：物理逻辑在区块生成前就已运行，导致玩家穿模。修复：引入 `isReady` 标志位并在 `findSafeSpawn` 成功后才放开物理限制。

## 4. 世界生成：树木悬空（“浮空林”事件）
* **症状**：树木浮在离地 1-3 格的空中。
* **原因与修复**：地形是由 3D 密度函数决定的，简单地在 `baseHeight` 处长树是不准确的。修复方法：引入**垂直向下扫描机制**，通过分析 `localData` 找到真实的实体表面 `surfaceY`。

## 5. 架构：异步闪现与数据冲突
* **症状**：挖掘方块后，方块又瞬间闪现回来。
* **原因与修复**：主线程发送构建请求后，Worker 返回结果过迟，此时主线程可能有更新的操作。修复方法：在主线程 Chunk 实例中维护 `lastRequestId`，Worker 回传时强制校验 `version` 是否一致。

## 6. 世界生成：地形/植被漂移（非确定性随机）
* **症状**：每次重新进入同一位置，树木的位置或地形细节会变化。
* **原因与修复**：生成算法中使用了 `Math.random()`。修复方法：替换为基于世界坐标 `(x, z)` 的 `pseudoRandom` 哈希函数，保证生成结果完全确定。

## 7. 数据存储：数据库 Delta 合并时机
* **症状**：保存了方块，但重新进入后方块丢失或只剩下地形。
* **原因与修复**：如果 Delta 合并在主线程合并网格时处理，会极度缓慢且易错。优化方案：将 Dexie 数据库中的增量修改 `deltas` 直接作为参数传给 Worker，在生成地形数组的最底层完成数据覆盖，实现一次性合并生成网格。

## 8. 音频：高频操作导致杂音/爆音
* **症状**：疯狂点击鼠标挖掘时，声音卡顿或浏览器报错。
* **原因与修复**：单个 `Audio` 对象不支持并行重叠播放。修复方案：引入 **Audio Object Pool (对象池)**，预加载 5-10 条音轨，通过 LRU 算法轮转使用，并辅以随机 `detune` 音调微调增强自然感。
