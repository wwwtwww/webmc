# 避坑指南与 Debug 日志 (Troubleshooting)

## 1. 渲染：深度排序错乱（透视/空间颠倒）
* **症状**：远处的方块渲染在近处的方块之上，画面类似“纪念碑谷”般错乱。
* **原因与修复**：确认 Three.js 材质设置中 `depthTest: true` 和 `depthWrite: true`。确保非玻璃/水方块的 `transparent` 属性为 `false`。

## 2. 渲染：单面破洞/面片丢失
* **症状**：大部分渲染正常，但特定方向（如某个侧面）看过去能直接看到天空背景。
* **原因与修复**：
    1.  **顶点环绕顺序错误**：WebGL 会剔除顺时针排列的三角形。检查生成该面 `indices` 时，两个三角形的顶点连接顺序必须是**逆时针 (CCW)**。
    2.  **相邻检测方向写反**：检查 `isAir(x+dx, y+dy, z+dz)` 中的偏移量是否与当前生成的面法线方向完全一致（避免 Copy-Paste 错误）。

## 3. 物理：玩家落地后疯狂抽搐/反复降落
* **症状**：玩家触地后视野剧烈抖动，或者后台坐标一直穿模后被拉回。
* **原因与修复**：碰撞检测触发修正坐标后，**忘记清零垂直动量**。必须在碰撞修正代码中加入 `velocity.y = 0`，否则累积的重力会在下一帧瞬间把玩家重新砸入地下。

## 4. 物理：玩家掉入虚空无限循环 (Respawn Loop)
* **症状**：玩家穿过地面一直往下掉，触发安全高度后重置到天上，继续掉落穿模。
* **排查方向**：重点检查从世界真实坐标 (World Pos) 映射到 Chunk 数组 1D 索引 (Index) 的数学转换公式。如果公式有误，底层碰撞检测查到的永远是 0 (空气)。
