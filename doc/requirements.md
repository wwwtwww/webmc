# 网页版体素引擎 (Web Voxel Engine) 需求文档

## 1. 项目概述
本项目旨在开发一款基于 Web 浏览器的轻量级 3D 体素游戏引擎（类似《我的世界》）。玩家可以在程序生成的无限世界中进行探索、挖掘和建筑。

## 2. 核心玩法
* **无限探索**：世界地形由 3D Simplex Noise 算法动态生成，支持悬崖、洞穴与多种生物群落。
* **生物群落 (Biomes)**：目前支持草地 (Grass)、沙漠 (Desert) 与雪山 (Snowy) 三种生态。
* **方块交互**：玩家可以破坏现有方块或放置新方块。支持 1-8 数字键切换不同材质。
* **物理系统**：
    * **生存模式**：具备 AABB 碰撞检测与重力感应。
    * **飞行模式 (God Mode)**：按 F 键切换，支持无碰撞自由穿梭。
* **持久化存储**：玩家对世界的修改会实时保存至浏览器 IndexedDB (Dexie.js)，重新进入时自动恢复。
* **沉浸音效**：包含环境音效、挖掘声、放置声及脚步声，支持音调随机化与多轨道叠加。

## 3. 操作说明
* **W, A, S, D**：控制玩家移动。
* **鼠标移动**：控制第一人称视角。
* **鼠标左键**：挖掘方块。
* **鼠标右键**：放置方块。
* **1 - 8 数字键**：切换快捷栏方块。
* **空格键 (Space)**：跳跃（生存）/ 上升（飞行）。
* **左 Shift 键**：下降（飞行）。
* **F 键**：切换飞行模式。
* **F3 键**：切换调试信息面板（显示坐标、FPS、生物群落）。
