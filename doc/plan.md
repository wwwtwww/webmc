# Minecraft Clone 项目计划书

## 1. 已完成功能 (Completed)

### 基础架构
- [x] **项目初始化**：使用 Vite + Three.js 搭建基础 ESM 开发环境
- [x] **文档规范 (NEW)**：建立 README.md 与项目开发文档规范
- [x] **世界高度 (16x256x16)**：支持 256 层高度的区块渲染与垂直加载
- [x] **网格化系统**：基于 Web Worker 实现异步网格化生成逻辑
- [x] **环境遮挡 (AO)**：实现基础的顶点环境光遮挡（AO）效果
- [x] **视锥剔除**：优化渲染性能，仅渲染摄像机视线内的区块
- [x] **数据同步机制**：支持基于 Request ID 的 Worker 通信确保数据一致性

### 持久化存储
- [x] **IndexedDB 存储**：集成 `Dexie.js` 实现玩家世界数据的持久化存储
- [x] **增量更新**：仅保存被修改过的区块数据 Deltas 以降低存储压力

### 交互系统
- [x] **物理碰撞**：实现基于 AABB 的玩家与世界方块碰撞检测
- [x] **控制台系统**：支持 T 键呼出控制台及 /time, /give, /tp 等调试命令
- [x] **飞行模式 (NEW)**：支持双击空格或按 F 切换飞行状态与 God Mode
- [x] **重力系统**：实现玩家的基础重力感应与落地判定逻辑
- [x] **调试面板 (F3/HUD)**：实时显示 FPS、坐标、区块坐标等调试信息
- [x] **准星射线**：使用 Raycaster 实现 5 格范围内的精准方块选取
- [x] **挖掘/放置**：支持方块破坏与放置，并同步更新周围 AO 效果
- [x] **安全生成 (Safe Spawn)**：确保玩家始终生成在非阻塞的地面之上

### 核心玩法
- [x] **怪物生成**：`MobManager` 自动在玩家周围生成僵尸。
- [x] **僵尸基础 AI**：简单的 3D 物理模拟和 FSM（跟随/闲逛）。
- [x] **路径规划 AI (NEW)**：为僵尸添加避障和寻路功能（基于 A* 算法）。
- [x] **伤害系统 (NEW)**：玩家可以攻击僵尸，僵尸也会攻击玩家。
- [x] **死亡与掉落**：实体死亡后消失并掉落经验球/物品。

### 物品与 UI
- [x] **背包系统 (Inventory)**：实现 36 格背包空间，支持 1-8 快捷栏切换
- [x] **2D 物品渲染**：支持在 UI 界面中以 2D 贴图形式展示物品图标
- [x] **合成系统 (2x2)**：实现基础的背包 2x2 合成逻辑与预览功能
- [x] **逻辑修复 (Bug Fixes)**：累计修复了 35 个 Bug，涵盖了寻路 AI 垂直移动、掉落物物理抖动、水下挖掘射线穿透、放置方块实体防穿模等核心问题。

### 地形生成
- [x] **随机地形**：集成 2D Simplex Noise 实现基础地形高度图生成
- [x] **无限世界**：支持随玩家移动动态加载与卸载区块数据
- [x] **AO 逻辑修复**：优化区块边界 AO 计算逻辑，消除渲染黑缝

### 音效系统
- [x] **音频管理 (AudioManager)**：实现基于 LRU 策略的 3D 空间音效系统
