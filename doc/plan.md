# Minecraft Clone 开发进度与计划

## 1. ✅ 已实现功能 (Completed)

### 渲染与核心系统
- [x] **基础框架**：基于 Vite + Three.js 的高效 ESM 开发环境。
- [x] **高度扩展 (16x256x16)**：支持 256 层垂直空间，彻底解决植被截断与宏观地形限制。
- [x] **渲染优化**：
    - **面剔除 (Face Culling)**：包含跨区块面剔除 (Cross-Chunk Culling)，相邻 Chunk 接触面不渲染。
    - **棋盘格变色**：基于坐标奇偶性的顶点变色，零开销分辨方块边界。
    - **多线程计算**：使用 Web Worker 处理 18x256x18 扩展数据块，主线程极致丝滑。
- [x] **视觉增强**：线性雾效果完美掩盖动态加载边界。

### 物理与交互
- [x] **双模式控制器**：
    - **生存模式**：AABB 碰撞检测、重力系统、滑动碰撞。
    - **飞行模式 (上帝视角)**：`F` 键切换，无视重力与碰撞 (Noclip)，支持垂直升降。
- [x] **交互系统**：
    - **射线检测**：支持跨区块的挖掘与放置。
    - **动态图标快捷栏**：CSS 3D 渲染立体图标，支持 1-7 数字键切换。
- [x] **调试增强 (F3/HUD)**：实现实时坐标 (XYZ)、区块 (Chunk)、FPS、群落 (Biome) 及移动模式的同步显示。

### 世界生成
- [x] **无限世界加载**：WorldManager 实时调度。
- [x] **差异化群落 (Biomes)**：
    - 引入基于 `biomeNoise` 的参数插值算法 (Lerp)。
    - **积雪高山**：高海拔、大起伏。
    - **温带平原**：中海拔、极平坦。
    - **热带沙漠**：低海拔、沙丘起伏。
- [x] **程序化植被**：确定性树木生成逻辑。

## 2. 🚧 待办任务 (To-Do List)

### 第一优先级：视觉细节与性能完善 (Current Focus)
- [ ] **环境光遮蔽 (Ambient Occlusion)**：在方块拐角处计算顶点颜色渐变，增强立体真实感。
- [ ] **方块透明度支持**：实现半透明材质层，支持真正的水体透视与透明玻璃。
- [ ] **多层级 LOD 优化**：针对远景区块使用更简单的网格生成逻辑，进一步提升大视距性能。

### 第二优先级：系统功能与玩法 (Future Enhancements)
- [ ] **持久化存储 (Save/Load)**：使用 localStorage 记录玩家对无限世界的修改。
- [ ] **多层噪声地形**：引入 3D 噪声实现悬崖、洞穴等更复杂的地形结构。
- [ ] **音效系统**：添加脚步声、方块破坏/放置声以及环境音效。

## 3. 下一步建议
建议下一步优先实现 **环境光遮蔽 (Ambient Occlusion)**。虽然目前有棋盘格变色，但方块交界处的转角阴影缺失仍会让画面略显单薄。通过基于周围 8 个方块状态计算顶点颜色加权，可以极大提升地形的体积感。
