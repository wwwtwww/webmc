# Minecraft Clone 开发进度与计划

## 1. ✅ 已实现功能 (Completed)

### 渲染与核心系统
- [x] **基础框架**：基于 Vite + Three.js 的高效 ESM 开发环境。
- [x] **体素引擎**：使用 `Uint8Array` 优化内存，支持 16x16x16 区块存储。
- [x] **渲染优化**：
    - **面剔除 (Face Culling)**：仅渲染暴露在空气中的面。
    - **棋盘格变色**：基于坐标奇偶性的顶点变色，清晰分辨方块边界。
    - **多线程计算 (Web Workers)**：将网格生成移至后台线程，确保极致丝滑。
- [x] **视觉增强**：
    - **线性雾 (Fog)**：掩盖远处区块的动态加载边界。
    - **多生物群落 (Biomes)**：实现沙漠、雪原、草地的自然分布。

### 物理与交互
- [x] **第一人称控制**：`PointerLockControls` 实现鼠标视角与 WASD 移动。
- [x] **物理碰撞**：实现 AABB 碰撞检测与重力系统，支持滑动碰撞与跳跃。
- [x] **交互系统**：
    - **方块交互**：左键挖掘，右键放置。
    - **图标化快捷栏**：纯 CSS 3D 图标，支持 1-9 数字键无缝切换。

### 世界生成
- [x] **无限世界**：`WorldManager` 自动根据玩家位置加载/销毁区块。
- [x] **无缝地形**：使用确定性 Simplex Noise 确保跨区块山脉连绵。
- [x] **程序化植被**：实现基于坐标哈希的确定性树木生成。

## 2. 🚧 待办任务 (To-Do List)

### 第一优先级：持久化与深度表现 (Current Focus)
- [ ] **持久化存储 (Save/Load)**：使用 `localStorage` 保存玩家对世界的挖掘和建造改动。
- [ ] **环境光遮蔽 (Ambient Occlusion)**：在方块拐角处计算模拟阴影，增强立体感。
- [ ] **方块透明度支持**：优化渲染管线，支持半透明水块或玻璃。

### 第二优先级：玩法扩展 (Future Enhancements)
- [ ] **UI 系统增强**：添加准星样式变换、坐标显示 (F3)。
- [ ] **更多地形细节**：添加洞穴生成系统、更多种类的植被（灌木、花朵）。
- [ ] **音效系统**：添加脚步声、环境背景音、方块破坏音效。

## 3. 下一步建议
建议下一步优先实现 **持久化存储 (Save/Load)**。随着无限世界和多群落的完善，玩家会产生更强的探索和建造欲望，保存建筑成果是提升游戏“粘性”的关键。
