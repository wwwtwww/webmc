- [x] **WebGL 指令压缩 (Bug 45)**：在 `Chunk._updateMeshGeometry` 中使用 `BufferAttribute.set()` 配合 `setDrawRange` 减少 GPU 内存上传量

## 3. 核心限制说明
- **3x3 渲染限制**：目前受限于单线程性能，最大渲染范围维持在 $3 \times 3$ 个区块
- **UI 交互**：需确保 Three.js 鼠标锁定模式与 DOM UI 点击不冲突
- **Worker 资源**：严禁在 Web Worker 中操作任何 DOM，所有 UI 数据必须序列化传输
