# 网页版体素引擎 (Web Voxel Engine) 需求文档

## 1. 项目概述
本项目旨在开发一款基于 Web 浏览器的轻量级 3D 体素游戏引擎。玩家可以在程序生成的无限世界中进行探索、挖掘和建筑。

* **核心玩法**：
    * **无限探索**：世界地形由 3D Simplex Noise 算法动态生成，支持悬崖、洞穴与多种生物群落。
    * **生物系统 (Mob System)**：
        * **僵尸 (Zombie)**：敌对生物。在夜晚或阴暗处生成，具备追踪玩家的 A* 寻路 AI。在日光下会自燃受损。
        * **猪 (Pig)**：被动生物。在草地生物群落生成，受到攻击会掉落生肉 (Raw Pork)。
        * **生成规则**：基于光照与生物群落的动态生成算法，支持最大数量限制。
* **生物群落 (Biomes)**：目前支持草地 (Grass)、沙漠 (Desert) 与雪山 (Snowy) 三种生态。
* **UI 与交互 (UI & Interaction)**：
    * **PlayerHUD**：实时显示玩家状态，包含生命值、饥饿度（预留）以及快速访问栏。
    * **背包与合成 (Inventory & Crafting)**：
        * **36 槽位背包**：支持物品堆叠、拿起与放下交互。
        * **2x2 随身合成**：支持基础物品（如原木变木板、木板变木棍）的背包内实时合成。
        * **3x3 工作台合成**：右键点击世界中的工作台可开启大九宫格合成区，支持合成复杂工具。
* **工具系统 (Tool System)**：
    * **工具类型**：包含木镐、木斧、木锹等基础工具。
    * **效能提升**：手持匹配工具挖掘对应方块时（如用斧头砍树），挖掘速度由默认的 1.5s 显著提升至 0.3s。
    * **采集等级**：某些坚硬方块（如石头）具备采收等级限制，徒手挖掘将不产生掉落物。
* **掉落物系统 (Item Drops)**：方块被挖掘或生物死亡后会产生掉落实体，具备物理碰撞体积与重力感应，支持向玩家身体中心的磁力吸附自动拾取。
* **方块交互**：
    * **连续挖掘**：长按鼠标左键可实现基于 **DDA 算法** 的高效射线追踪 (Raycasting) 连续破坏，具备 200ms 的视角容错缓冲。
    * **范围限制**：最大交互距离限制为 5 格 (MAX_REACH)。
    * **快速放置**：鼠标右键立即放置方块。支持 1-8 数字键切换。
* **底层优化**：
    * **多线程架构**：利用 Web Worker 处理地形生成，通过 **SharedArrayBuffer** 实现主线程与工作线程间的零拷贝内存共享，大幅提升渲染流畅度。
* **物理系统**：具备 AABB 碰撞检测与重力感应。
* **开发者工具**：
    * **控制台指令**：支持 `/time`、`/give`、`/tp`、`/spawn [zombie|pig]` 以及 `/clear-mobs` 指令进行快捷调试。
    * **动态环境**：支持通过指令即时切换场景的天空氛围。
* **持久化存储**：玩家对世界的修改会实时保存至浏览器 IndexedDB。
* **沉浸音效**：包含挖掘声、放置声及脚步声。

## 3. 操作说明
* **W, A, S, D**：控制玩家移动。
* **鼠标移动**：控制第一人称视角。
* **鼠标左键 (长按)**：挖掘方块。
* **鼠标右键**：放置方块。
* **E / Tab**：打开/关闭背包界面。
* **T / ` (波浪号)**：呼出开发者指令控制台。
* **1 - 8 数字键**：切换快捷栏方块。
* **空格键 (Space)**：跳跃。
* **F**：切换飞行模式 (GOD MODE)。
* **F3 键**：切换调试信息面板（显示坐标、FPS、生物群落）。
